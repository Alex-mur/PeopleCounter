# PeopleCounter 🚦

**PeopleCounter** — это распределенная система подсчета посетителей в зонах интереса на основе видеопотоков, использующая алгоритмы компьютерного зрения и микросервисную архитектуру.

Система состоит из трех основных компонентов:
1. **Manager (Backend)** — центральный сервер управления на Go.
2. **WorkerYOLO26 (AI/ML Service)** — сервис видеоаналитики на Python (YOLO).
3. **Front (Frontend)** — пользовательский веб-интерфейс на Vue 3.

---

## 🏗 Архитектура проекта

### 1. ⚙️ Manager (`PeopleCounter_Manager`)
Центральный управляющий сервис, написанный на **Go 1.24**.
- **Функции:** Управление счетчиками посетителей, сбор статистики, агрегация данных и аутентификация пользователей.
- **Технологии:** Go-chi, PostgreSQL (через pgx), JWT Auth.
- **API:** REST API с поддержкой Swagger/OpenAPI (доступно по умолчанию на порту `9000`).

### 2. 🧠 Worker (`PeopleCounter_WorkerYOLO26`)
Веб-сервис для обработки видеопотоков и детекции людей, написанный на **Python**.
- **Функции:** Захват видео (IP-камеры, RTSP, видеофайлы), детекция людей с использованием моделей YOLO, подсчет пересечений заданных линий.
- **Технологии:** OpenCV, Ultralytics YOLO, FastAPI/Flask, DuckDB (для аналитики и локального хранения).
- **Особенности:** Поддержка Multistreaming, автоматический перезапуск при разрыве соединения с камерой, автоочистка логов.

### 3. 🖥 Frontend (`PeopleCounter_Front`)
Пользовательский интерфейс, разработанный на **Vue 3** (Composition API) и Vite.
- **Функции:** Визуализация статистики подсчета, управление сервисами, настройка линий подсчета, панель авторизации (Login, Management, Statistics, ServiceCounters).
- **Технологии:** Vue 3, Vue Router, Pinia, Nginx (для продакшен сборки).

## 🚀 Установка и запуск

Проект спроектирован для запуска через **Docker** и **Docker Compose**. Каждый компонент имеет собственный `docker-compose.yml` и конфигурационные файлы [code_file:11].

### Предварительные требования:
- Установленный Docker и Docker Compose.
- Настроенная база данных PostgreSQL (для менеджера).

### Быстрый старт:

**1. Manager:**
```bash
cd PeopleCounter_Manager
# Скопируйте и настройте конфигурацию
cp example.config.json config.json 
# Заполните config.json актуальными данными для подключения к БД
docker-compose up -d --build
```

**2. Worker:**
```bash
cd PeopleCounter_WorkerYOLO26
# Создайте конфигурационный файл
cp config.ini.example config.ini
# Настройте видеопотоки и зоны в config.ini
docker-compose up -d --build
```

**3. Frontend:**
```bash
cd PeopleCounter_Front
docker-compose up -d --build
```
*Frontend по умолчанию будет доступен на порту `8080` [code_file:11].*
